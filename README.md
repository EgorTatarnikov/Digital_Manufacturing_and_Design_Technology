# Digital Manufacturing & Design Technology

Репозиторий с материалами по специализации **Digital Manufacturing & Design Technology**.

Источник специализации:  
https://www.coursera.org/specializations/digital-manufacturing-design-technology

## О репозитории

В этом репозитории собраны материалы по всей специализации в удобном для быстрого изучения и повторения формате.

Здесь хранятся:

- **субтитры лекций**
- **сжатые структурированные конспекты**
- **сертификаты о прохождении специализации**
- **промпт**, который использовался для преобразования лекций в плотные конспекты

Основная идея репозитория — **сохранить материалы в публичном и быстро доступном виде**, чтобы к ним можно было легко возвращаться без необходимости заново искать лекции, пересматривать длинные видео или извлекать ключевые мысли вручную.

## Особенности конспектов

Конспекты в этом репозитории создавались с упором на:

- высокую плотность полезной информации
- минимум воды и повторов
- сохранение терминологии
- выделение основных идей, определений и связей между понятиями
- удобство для повторения, поиска и быстрого восстановления темы в памяти

Это не пересказ “в свободной форме”, а попытка превратить лекционный материал в **компактные учебные заметки**, пригодные для быстрого доступа к сути.

Дополнительно:

## Для чего это может быть полезно

Репозиторий может быть полезен:

- студентам и инженерам, изучающим **Industry 4.0 / Digital Manufacturing**
- тем, кто хочет быстро повторить материал без просмотра всех лекций
- тем, кто ищет структурированную терминологическую базу по курсам специализации
- тем, кто интересуется применением LLM/AI для переработки учебных материалов в качественные конспекты

## Формат материалов

Внутри папок могут встречаться:

- текстовые файлы с субтитрами лекций
- конспекты по отдельным темам
- обобщённые материалы по курсам
- вспомогательные файлы, связанные с обработкой и структурированием контента

## Зачем здесь лежит Prompt.txt

Файл `Prompt.txt` добавлен в репозиторий как часть методологии работы с материалом.

Он показывает, каким образом лекционный текст преобразовывался в:

- сжатый
- логически структурированный
- терминологически насыщенный
- удобный для повторения конспект

Это полезно, если вы хотите:
- воспроизвести такой же подход для других курсов
- адаптировать промпт под свои задачи
- использовать репозиторий не только как архив материалов, но и как пример пайплайна обработки учебного контента

## Навигация

Рекомендуемый способ использования репозитория:

1. выбрать нужный курс по папке
2. открыть субтитры или конспект интересующей темы
3. использовать конспекты для быстрого повторения ключевых понятий и идей
4. при необходимости посмотреть `Prompt.txt`, чтобы понять логику преобразования лекций в структурированный материал
